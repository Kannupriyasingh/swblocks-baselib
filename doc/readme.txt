"<project>.doxygen" files contains Doxygen settings in order for various code documenation to be generated.
To generate and view the documenation:
* Download the tool from http://www.stack.nl/~dimitri/doxygen/download.html, 64-bit version ZIP and unpack it
* Make the 'doc' folder where this readme file is the current folder in the console
* Run: md output
* Run: doxygen.exe <project>.doxygen
* Open output\<project>\html\index.html
* In order to use the search in IE, click the warning strip and "Allow blocked content"
